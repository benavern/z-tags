What is Z-Tags?

Z-Notif is a javascript plugin that lets you create tags areas very easily. By using it, you will be able to have tags sections on your webpage, and will have the possibility to let users adding tags to an area.

You won't need any javascript library, it is totally written in vanillaJs.

Demo

Here you can see a default tags section. There are 3 tags predefined, you can add and delete tags in it. There is also a method to get the data in it.
